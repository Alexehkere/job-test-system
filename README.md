# Система тестирования соискателей при приёме на работу

## Описание
Веб-приложение для автоматизации тестирования кандидатов на этапе собеседования.  
Работодатели создают тесты, назначают их соискателям, получают результаты.  
Администраторы управляют доступом работодателей.

---

## Функционал

### Для **соискателей** (`applicant`)
- Регистрация и вход
- Просмотр назначенных тестов
- Прохождение тестов с таймером
- Просмотр результатов

### Для **работодателей** (`employer`)
- Создание тестов и вопросов (один/много/открытый ответ)
- Назначение тестов соискателям
- Просмотр результатов

### Для **администраторов** (`admin`)
- Одобрение/отклонение заявок работодателей
- Блокировка/разблокировка пользователей
- Управление приглашениями соискателей

---

## Типы вопросов
- **Один правильный ответ** (`single`)
- **Несколько правильных** (`multiple`)
- **Открытый ответ** (`open`)
