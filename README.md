# Система тестирования соискателей при приёме на работу

## Описание
Веб-приложение для автоматизации тестирования кандидатов. Работодатели создают тесты, назначают их соискателям, получают результаты.

## Функции
- Регистрация: соискатель, работодатель, админ
- Создание тестов с вопросами (один/много/открытый ответ)
- Назначение тестов
- Прохождение тестов с таймером
- Управление пользователями (одобрение, блокировка)
- Уведомления

## Технологии
- Django 5.2
- Bootstrap 5
- SQLite (в разработке)

python manage.py runserver
